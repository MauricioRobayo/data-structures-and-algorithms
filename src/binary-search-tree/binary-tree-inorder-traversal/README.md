## Binary Tree In-order Traversal

Given a binary tree, return the in-order traversal of its nodes' values.

Example:

```
Input: [1,null,2,3]
   1
    \
     2
    /
   3
```

Output: [1,3,2]

**Follow up**: Recursive solution is trivial, could you do it iteratively?
